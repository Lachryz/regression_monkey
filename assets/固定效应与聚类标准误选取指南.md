# 固定效应与聚类标准误选取指南

## 一、核心原则

**固定效应**：消除遗漏变量偏误，影响点估计 $\hat\beta$。
在哪个维度存在系统性遗漏变量，就在那个维度加固定效应；但该维度不能与处理变量的分配维度完全重合，否则识别失效。

**聚类标准误**：修正残差相关结构对 SE 的低估，影响推断，不影响 $\hat\beta$。
残差在哪个维度有系统性相关，就在那个维度聚类；通常与处理变量的分配维度一致。

**两者独立判断，互不决定。**

---

## 二、固定效应选取规则

1. 固定效应只能吸收**时不变**部分；时变混淆需用交互项（如行业×年度）。
2. 维度越细控制越彻底，但处理变量须在该维度内仍有 within 变异。
3. **红线**：固定效应维度 = 处理变量分配维度 → 识别失效，处理变量 within 变异归零。
4. 加入高维固定效应后，检查处理变量的 within 标准差是否接近零；若是，退回更粗的设定。

---

## 三、聚类标准误选取规则

1. 聚类维度 = 残差真实相关所在维度，通常 = 处理变量分配维度。
2. 固定效应吸收了某维度的均值，但未消除该维度内的残差相关，两者独立判断。
3. 聚类维度不能比残差真实相关的层面更细，否则 SE 低估。
4. **经验阈值**：聚类组数 < 50 时渐近理论不可靠，须同时报告 wild cluster bootstrap 结果。
5. `robust` 仅解决异方差，假设观测间独立；面板数据中个体重复出现，残差必然存在组内相关，应直接使用 `cluster` 而非 `robust`。

---

## 四、常见组合

```stata
* 标准设定：公司时变冲击
absorb(firm year) vce(cluster firm)
```

个体+年度 FE，公司聚类，会计实证的默认基准。

```stata
* 控制行业时变干扰
absorb(firm i.industry#i.year) vce(cluster firm)
```

行业×年度 FE 替代年度 FE，吸收行业景气和行业监管冲击；处理变量必须是公司层面异质性冲击而非行业统一政策。

```stata
* 控制地区时变干扰
absorb(firm i.region#i.year) vce(cluster firm)
```

地区×年度 FE 替代年度 FE，吸收地方政策和区域经济周期；处理变量必须是公司层面异质性冲击而非地区统一政策。

```stata
* 同时控制行业和地区时变干扰（最严格）
absorb(firm i.industry#i.year i.region#i.year) vce(cluster firm)
```

双重剥离行业和地区时变冲击；通常用于稳健性检验，处理变量须是公司特有冲击。

```stata
* 地区层面政策
absorb(firm year) vce(cluster region)
```

处理变量在地区层面分配，区内所有公司共享同一处理状态，必须在地区层面聚类；地区组数少时建议同时报告 wild cluster bootstrap 结果。

```stata
* 行业层面政策
absorb(firm year) vce(cluster industry)
```

处理变量在行业层面分配，同行业公司处理状态相同；行业数量少时同样面临组数不足问题。

```stata
* 短面板或无法追踪个体
absorb(industry region year) vce(cluster industry)
```

无个体层面重复观测，用行业+地区+年度三向 FE 替代；内生性控制力度最弱，因果解释须有其他保障。

```stata
* 双向聚类
absorb(firm year) vce(cluster firm year)
```

同时担心公司层面跨期相关和年度层面截面相关；两个维度组数须均足够，否则退回单向聚类。

## 五、判断流程

```md
第一步：处理变量在哪个层面分配？是否时变？
        ↓
第二步（固定效应）：哪些维度存在遗漏变量？
        时不变遗漏变量  → 加对应维度 FE
        时变遗漏变量    → 加交互 FE（维度不能与处理变量分配维度相同）
        ↓
第三步（聚类）：残差在哪个层面有系统性相关？
        通常 = 处理变量分配维度 → 在该层面聚类
        组数 < 50 → 同时报告 wild cluster bootstrap
        ↓
第四步（诊断）：处理变量的 within 变异是否仍然存在？
        within 标准差接近零 → 过度控制，退回更粗的固定效应设定
```

---

## 六、维度粗细的方向性规律

| | 固定效应 | 聚类标准误 |
| --- | --- | --- |
| 维度从细到粗 | 控制力度减弱（个体 > 行业 > 地区） | SE 更大、推断更保守（公司 < 行业 < 地区） |
| 目标 | 消除遗漏变量，净化点估计 | 承认残差相关，修正推断 |
| 威胁 | 过细导致识别失效 | 过细导致 SE 低估；组数过少导致渐近失效 |

固定效应和聚类的粗细方向相反：固定效应维度越细越严格，聚类维度越粗越保守。两者都从变异所在的层面出发判断，但服务于不同的目标。
